Onboard

Onboard class used to start, control and end app walkthrough experience

new Onboard(options: OnboardOptions)
Parameters
options (OnboardOptions)
Example
const onboard = new Onboard({
  nextText:"Next"
});

onboard.start([
		{
			selector:".menu-element",
			title:"Step 1 title",
			text:"Step 1 text."
			
		},
		{
			selector:".sidebar-element",
			title:"Step 2 title",
			text:"Step 2 text."
			
		}
	]);

OnboardOptions

Options for the main Onboard instance

OnboardOptions

Type: Object

Properties
fillColor (string) : color of the modal background, defaults to FILL_COLOR
fillOpacity (number) : opacity of the modal background, defaults to FILL_OPACITY
nextText (string) : default text for the Next button, defaults to NEXT_TEXT
windowClassName (string) : custom class name for popup window, defaults to WINDOW_CLASS_NAME
animationDuration (number) : lenght of animation duration when transitioning between steps, defaults to ANIMATION_DURATION
steps (Array<StepOptions>) : definition of steps

StepOptions

Options for the each Onboard step

StepOptions

Type: Object

Properties
selector ((string | HTMLElement)) : of spotlighted HTML/SVG node(s), if no selector is provided, no element is spotlighted and window is positioned in the center
window (string) : title for this step, can be HTML
body (string) : text for this step, can be HTML
nextText (string) : text for the Next button, defaults to NEXT_TEXT or OnboardOptions
showProgress (boolean) : hides progress bar for this step if false
windowClass (string) : name allows to set additional window class name for this step, this can be used for customization
windowWidth (number) : width of window for this step
shape (ShapeOptions) : shape options

ShapeOptions

Shape otions for the each step

ShapeOptions

Type: Object

Properties
radius (number) : of the shape, can be customized if target element is rectangle but spotlight is required with rounded corners
type (string) : can be set to change shape's type, supports 'rectangle' (default) and 'circle'.
width (number) : width of the shape if it requires to be different than target element's width
height (number) : height of the shape if it requires to be different than target element's height
strokeWidth (number) : can be set to extend spotlight by given number
